  • Выкапываем котлован для обустройства скважины с кессоном, либо копаем приямок для обустройства с адаптером и траншею по необходимости
  • Обрезаем обсадную трубу в нижней точке при монтаже кессона,
    при монтаже скважинного адаптера труба не обрезается
  • Производим монтаж кессона либо врезку в трубу скважинного адаптера
  • Монтируем насосное оборудование в скважине
  • В кессонной ёмкости устанавливаем оголовок, гидроаккумулятор, реле давления, автоматику, монтируем летний поливочный кран
  • Для прокладки водоподающей трубы роем траншею глубиной ниже точки промерзания почвы и не менее 1,6 метра
  • В дом заводим две магистрали, одна водоподающая, вторая для подводки электрокабеля от насоса, при необходимости устанавливается греющий кабель
  • Устанавливаем фильтры для очистки воды
  • Подключаем оборудование к электричеству, запускаем системы, тестируем
